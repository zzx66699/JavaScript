# Number
## Math
```js
let x = 4.1;
let y = 5;
let z = 9;

let maximum = Math.max(x, y, z);

let minimum = Math.min(x, y, z);

x = Math.round(x);
console.log(x);
>>> 4

// down round
x = Math.floor(x);
console.log(x);
>>> 4

// up round
x = Math.ceil(x);
console.log(x);
>>> 5

// Exponentiation
x = Math.power(x, 2);

// Square root
x = Math.sqrt(x);

// Distance from 0 (absolute value)
x = Math.abs(x);

// Built-in constants
x = Math.PI;

// Generate a random number between 0 and 1
x = Math.random();
```
